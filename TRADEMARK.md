# Ferreus Vault Trademark Policy

## Overview

“Ferreus Vault” is a trademark of the Ferreus Vault project.

This trademark policy exists to:

- Protect users from malicious or misleading software
- Preserve trust in a security-critical application
- Allow open-source development without brand abuse

The source code is licensed under the GNU GPLv3.
The name and branding are NOT.

---

## What You MAY Do

You may, without permission:

- Fork the Ferreus Vault source code
- Modify the code for personal or internal use
- Distribute modified versions under the GPLv3
- Refer to Ferreus Vault factually (e.g., “based on Ferreus Vault”)

---

## What You MAY NOT Do (Without Written Permission)

You may NOT:

- Use the name **“Ferreus Vault”** for a forked or modified version
- Use the Ferreus Vault logo or branding
- Present a modified version as an official release
- Imply endorsement or affiliation with the Ferreus Vault project

This includes:

- App names
- Website names
- Package names
- Marketing material

---

## Required Naming for Forks

Forks must use a **clearly distinct name**, for example:

- “IronSafe (forked from Ferreus Vault)”
- “FV-Derived Vault”

This requirement exists to prevent user confusion and supply-chain attacks.

---

## Rationale (Security Context)

Ferreus Vault is a password vault.
User trust depends on knowing **exactly** what software they are running.

Trademark restrictions prevent:

- Trojanized forks
- Backdoored builds
- Impersonation attacks
